# sleepy-coder-2

Explainer video for [sleepy-coder](https://github.com/softwarewrighter/sleepy-coder) - a continual learning agent that fixes Rust code errors and learns from mistakes overnight using Parameter-efficient Continual Training (PaCT).

## Video Concept

Target: 2-3 minute landscape explainer video for Behind the Build series.

### Key Topics
- Day/night learning cycle metaphor
- Rust borrow checker challenges as training data
- LoRA-based continual learning
- Evaluation tracking and improvement metrics
